This is an an ASM version of VoyageGame (https://replit.com/@JesseKane/VoyageGame#main.py), a simple text based game I wrote in freshman year

Except this time its in assembly. NOTE: This program adheres to the 64-bit SystemV ABI, and uses AT&T Assembly Syntax. It is designed to be assembled by the GNU assembler, and links to the C standard library
